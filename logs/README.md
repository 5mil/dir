# Technician Logs

All technician continuation logs for project `dir` are stored here.

## Folder Structure

```
logs/
  README.md              ← this file (index)
  {YEAR}/
    {Month}/
      TECHNICIAN_LOG.md  ← monthly append-only log
```

## Convention

- One `TECHNICIAN_LOG.md` per calendar month
- Entries are **append-only** — never overwrite, never delete
- Each entry carries: Entry number, Stardate, timestamp (EDT), context summary, full log text
- New month = new file in new folder, cross-linked from this index
